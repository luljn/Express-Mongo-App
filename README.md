# Express-Mongo-App-Backend

## Description

This is the backend code (API) for a web app built with Angular, Express and MongoDB.

## Tools used 

* [Node.js ![Node.js](https://skillicons.dev/icons?i=nodejs "Node.js")](https://nodejs.org/en)
* [Express ![Express](https://skillicons.dev/icons?i=express "Express")](https://expressjs.com/)
* [MongoDB ![MongoDB](https://skillicons.dev/icons?i=mongo "MongoDB")](https://www.mongodb.com/)

## Getting started

* Run `npm install` then `npm run start` at the root of the project.
* The routes are `/api/auth` and `api/stuff`.
